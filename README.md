# Todo List App

Aplikasi Todo List sederhana yang dibuat menggunakan Angular 20.
Aplikasi ini memiliki 1 halaman utama yaitu halaman Todo List.

## Fitur

- Menambahkan todo baru
- Menghapus todo
- Toggle status todo selesai / belum selesai
- Search todo berdasarkan judul
- Sorting todo
- Validasi form dengan Reactive Forms
- Responsive UI dengan TailwindCSS

## Teknologi yang Digunakan

- [Angular 20](https://angular.dev/)
- [RxJS](https://rxjs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [TailwindCSS](https://tailwindcss.com/)

## Cara Instalasi

Ikuti petunjuk dibawah ini untuk menjalankan aplikasi secara lokal

```bash
# 1. Clone repository
git clone https://github.com/username/todo-list-app.git
cd todo-list-app

# 2. Install dependencies
npm install

# 3. Jalankan aplikasi
ng serve

# 4. Buka di browser
http://localhost:4200
```
